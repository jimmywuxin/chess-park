# Changelog

## v1.2.0 (2026-05-28)

### AI 引擎全面升级
- **置换表（Transposition Table）** — 缓存已评估棋局，避免重复计算
- **迭代加深搜索** — 从 1 层逐步加深到 maxDepth，配合限时搜索，超时返回当前最优
- **静默搜索（Quiescence Search）** — 叶子节点继续搜索吃子走法，消除视野效应
- **空着裁剪（Null Move Pruning）** — 跳过己方走棋测试局面，大幅减少搜索节点
- **杀手走法（Killer Move Heuristic）** — 每层记录产生截断的走法，优先搜索
- **历史启发式（History Heuristic）** — 全局统计走法效果，优化排序
- **Late Move Reduction（LMR）** — 靠后的非吃子走法减少搜索深度
- **开局库** — 从 itlwei/chess 移植 100+ 套开局棋谱
- **完整位置评分表** — 从 itlwei/chess 移植车马炮兵相士将完整 10×9 位置价值表

### 难度参数调整
- 简单：4 层 / 2 秒
- 中等：6 层 / 4 秒
- 困难：8 层 / 8 秒

### 走棋动画
- 玩家走棋 600ms、AI 走棋 1200ms，双独立动画控制器
